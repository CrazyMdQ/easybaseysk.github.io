如何製作電子版答案之書

本教學將引導您從零開始，製作一個簡單的電子版答案之書網頁。這個網頁可以隨機顯示預設的答案，也可以手動切換。

1. HTML 結構

首先，我們來建立網頁的基本 HTML 結構。這包括頁面標題、內容顯示區域和控制按鈕。

<!DOCTYPE html> <!-- 宣告文件類型為 HTML5 -->
<html lang="zh-Hant"> <!-- 設定網頁語言為繁體中文 -->
<head>
    <meta charset="UTF-8"> <!-- 設定字元編碼為 UTF-8，確保中文顯示正常 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- 設定視口，使網頁在不同設備上響應式顯示 -->
    <title>電子版答案之書</title> <!-- 設定網頁標題，顯示在瀏覽器標籤頁上 -->
    <!-- CSS 樣式將在這裡引入 -->
</head>
<body>
    <div style="background-color: white; width: 100%; height: 80px;padding: 10px; position: fixed; top: 0; left: 0; z-index: 1000;"> 
        <!-- 頂部固定導航欄，包含返回主頁的圖標 -->
        <div class="home-icon" onclick="goHome()"><img src="../icon/光.png" style="width: 75px;height: 75px;"></div> 
        <!-- 點擊圖標時觸發 goHome() 函數，返回主頁 -->
    </div> 
    <br> <!-- 換行，用於調整佈局 -->
    <div class="container"> <!-- 主要內容容器，用於包裹答案之書的內容 -->
        <h1>電子版答案之書</h1> <!-- 網頁主標題 -->
        <div id="content">歡迎使用電子版答案之書！請點擊按鈕開始探索。</div> <!-- 顯示答案內容的區域 -->
        <div class="btn-group"> <!-- 按鈕組，包含上一頁、隨機和下一頁按鈕 -->
            <button class="btn-prev" onclick="changePage('pre')">上一頁</button> <!-- 上一頁按鈕，點擊時觸發 changePage('pre') -->
            <button class="btn-random" onclick="changePage('random')">隨機</button> <!-- 隨機按鈕，點擊時觸發 changePage('random') -->
            <button class="btn-next" onclick="changePage('next')">下一頁</button> <!-- 下一頁按鈕，點擊時觸發 changePage('next') -->
        </div>
    </div>

    <!-- JavaScript 程式碼將在這裡引入 -->
</body>
</html>

2. CSS 樣式

接下來，我們為網頁添加樣式，使其看起來更美觀。您可以將以下 CSS 程式碼放入 <head> 標籤內的 <style> 標籤中。

body {
    font-family: Arial, sans-serif; /* 設定字體 */
    background-color: #f8f9fa; /* 設定背景顏色 */
    padding: 20px; /* 設定內邊距 */
    display: flex; /* 使用 Flexbox 佈局 */
    justify-content: center; /* 水平居中內容 */
    align-items: center; /* 垂直居中內容 */
    min-height: 100vh; /* 最小高度為視口高度 */
    margin: 0; /* 移除外邊距 */
}
.container {
    width: 600px;  /* 固定寬度 */
    height: 400px;  /* 固定高度 */
    background: white; /* 背景為白色 */
    padding: 2rem; /* 內邊距 */
    border-radius: 10px; /* 圓角邊框 */
    box-shadow: 0 4px 8px rgba(0,0,0,0.1); /* 陰影效果 */
    text-align: center; /* 文字居中 */
    overflow: auto;  /* 添加滾動條以防內容溢出 */
    position: relative; /* 添加相對定位以便放置主頁圖標 */
}
.home-icon {
    position: fixed; /* 固定定位 */
    top: 10px; /* 距離頂部 10px */
    left: 10px; /* 距離左側 10px */
    font-size: 24px; /* 字體大小 */
    cursor: pointer; /* 鼠標懸停時顯示手型 */
    color: #007bff; /* 顏色 */
    z-index: 1000; /* 層級 */
}
.home-icon:hover {
    color: #0056b3; /* 鼠標懸停時顏色變深 */
}
h1 {
    color: #333; /* 標題顏色 */
    margin-bottom: 1.5rem; /* 底部外邊距 */
}
#content {
    min-height: 150px; /* 最小高度 */
    padding: 1rem; /* 內邊距 */
    margin: 1rem 0; /* 上下外邊距 */
    border: 1px solid #ddd; /* 邊框 */
    border-radius: 5px; /* 圓角 */
    background-color: #fafafa; /* 背景顏色 */
    position: relative; /* 相對定位 */
}
.page-number {
    position: absolute; /* 絕對定位 */
    bottom: 5px; /* 距離底部 5px */
    right: 10px; /* 距離右側 10px */
    color: #666; /* 顏色 */
    font-size: 0.9rem; /* 字體大小 */
}
.btn-group {
    display: flex; /* 使用 Flexbox 佈局 */
    justify-content: center; /* 水平居中 */
    gap: 1rem; /* 元素間距 */
    
    margin-top: 1.5rem; /* 頂部外邊距 */
}
button {
    padding: 0.5rem 1rem; /* 內邊距 */
    border: none; /* 無邊框 */
    border-radius: 5px; /* 圓角 */
    cursor: pointer; /* 鼠標懸停時顯示手型 */
    font-size: 1rem; /* 字體大小 */
    transition: background-color 0.3s; /* 背景顏色過渡效果 */
}
button:hover {
    opacity: 0.9; /* 鼠標懸停時透明度降低 */
}
.btn-prev {
    background-color: #007bff; /* 背景顏色 */
    color: white; /* 文字顏色 */
}
.btn-next {
    background-color: #007bff; /* 背景顏色 */
    color: white; /* 文字顏色 */
}
.btn-random {
    background-color: #28a745; /* 背景顏色 */
    color: white; /* 文字顏色 */
}

document.getElementById('content').innerHTML = `
    ${pages[currentPage]} // 顯示當前頁面的內容
    <div class="page-number">第 ${currentPage + 1} 頁</div> // 顯示頁碼
`;

function changePage(action) { // 定義切換頁面的函數
    if (action === 'pre') { // 如果動作是 'pre' (上一頁)
        currentPage = Math.max(0, currentPage - 1); // 頁碼減 1，但不小於 0
    } else if (action === 'next') { // 如果動作是 'next' (下一頁)
        currentPage = Math.min(pages.length - 1, currentPage + 1); // 頁碼加 1，但不超過最大頁碼
    } else if (action === 'random') { // 如果動作是 'random' (隨機)
        currentPage = Math.floor(Math.random() * (pages.length - 1)) + 1; // 生成一個隨機頁碼
    }
    document.getElementById('content').innerHTML = `
        ${pages[currentPage]} // 更新顯示內容
        <div class="page-number">第 ${currentPage + 1} 頁</div> // 更新頁碼顯示
    `;
}
